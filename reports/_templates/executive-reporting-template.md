# Executive Reporting Template

**Assessment Type:** Simulated Web Pentest Lab  
**Environment:** Controlled educational environment  
**Audience:** Technical stakeholders / portfolio reviewers  

## Executive Summary
Provide a short, decision-oriented summary of the assessed scenario, the main risk observed, and the mitigation direction.

## Scope
- Application component
- Vulnerable implementation
- Fixed implementation
- Browser/client-side controls involved

## Methodology
- Manual testing
- Input manipulation
- DOM inspection
- Authorization review
- Session and storage review
- Validation of mitigation

## Findings
Each finding should include:
- Title
- Severity
- Description
- Technical details
- Proof of concept
- Impact
- Remediation
- Validation after fix

## Notes
Use this template only for simulated lab and portfolio scenarios, not for production claims.
